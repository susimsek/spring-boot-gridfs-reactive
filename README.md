## Prerequisites

* Java 11
* Maven 3.3+
* Docker 19.03+
* Docker Compose 1.25+

## Installation

```sh
./mvnw compile jib:dockerBuild
```

```sh
docker-compose up -d 
```

## Used Technologies

* Spring Boot 2.4.1
* Mongodb
* Spring Boot Web Flux
* Spring Boot Actuator
* Lombok
* Dev Tools
* Spring Boot Test
